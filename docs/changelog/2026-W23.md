## Breaking Changes

- **임베딩 모델 업그레이드**: `gemini-embedding-001` → `gemini-embedding-2`
  - 입력 한도 2,048 → 8,192 토큰 (비용 동일, 무료)
  - DB 임베딩 전량 리셋 + 재생성 완료 (92건)
  - 재생성 직후 챗봇 검색 품질 일시 변동 가능 → embed-pending cron이 자동 처리

- **clubs.active 컬럼 삭제** (migration 050)
  - `status` 컬럼만 사용. 코드에서 `active` 참조 시 에러 발생
  - clubs-approve에서 active 설정 라인 제거됨
