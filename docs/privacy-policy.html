개인정보처리방침

매치포인트 (Match Point)

시행일: 2026년 3월 1일

매치포인트(이하 "앱")은 이용자의 개인정보를 소중히 여기며, 「개인정보 보호법」 등 관련 법령을 준수합니다. 본 개인정보처리방침은 앱이 수집하는 정보, 사용 목적, 보관 기간 및 이용자의 권리에 대해 설명합니다.

1. Information We Collect — 수집하는 정보

앱은 서비스 제공을 위해 아래의 정보를 수집합니다.

수집 항목 수집 방법 필수 여부
이메일 주소, 비밀번호 회원가입 시 이용자 직접 입력 필수
채팅 메시지 챗봇 이용 시 자동 저장 필수
사용자 설정
(선호 지역, 알림 설정)		 이용자 직접 설정 선택
기기 푸시 토큰 알림 허용 시 자동 수집 선택
수집하지 않는 정보: 위치 정보, 연락처, 결제 정보, 사진 및 영상. 스피드건 기능에서 사용하는 영상은 기기 내에서만 처리되며 서버에 전송되지 않습니다. 앱에는 광고 SDK, 분석 SDK, 사용자 추적 도구가 포함되어 있지 않습니다.

2. How We Use Your Information — 정보 사용 목적

수집된 정보는 다음의 목적으로만 사용됩니다.

  • 계정 생성 및 인증: 이메일과 비밀번호를 통한 로그인 및 본인 확인
  • AI 챗봇 서비스 제공: 이용자의 질문에 대해 테니스 대회 정보를 검색하고 답변 생성
  • 대화 이력 제공: 이전 채팅 내용을 다시 확인할 수 있도록 저장
  • 맞춤 정보 제공: 선호 지역 설정에 따른 관련 대회 정보 우선 안내
  • 푸시 알림: 대회 일정 등 관련 알림 전송 (이용자가 허용한 경우에 한함)

3. Data Retention — 정보 보관 기간

정보 유형 보관 기간 파기 방법
채팅 메시지 90일 후 자동 삭제 데이터베이스에서 영구 삭제
계정 정보 (이메일, 비밀번호) 회원 탈퇴 시 즉시 삭제 데이터베이스에서 영구 삭제
사용자 설정 회원 탈퇴 시 즉시 삭제 데이터베이스에서 영구 삭제
기기 푸시 토큰 회원 탈퇴 시 또는 알림 해제 시 삭제 데이터베이스에서 영구 삭제

관련 법령에 의해 보존이 필요한 경우, 해당 법령에서 정한 기간 동안 별도 보관 후 파기합니다.

4. Third-Party Services — 제3자 제공

앱은 서비스 운영을 위해 다음의 제3자 서비스를 이용합니다.

서비스 제공 업체 제공 정보 이용 목적
Supabase Supabase Inc. 이메일, 비밀번호(암호화), 채팅 메시지, 사용자 설정, 푸시 토큰 사용자 인증 및 데이터 저장
Google Gemini API Google LLC 채팅 메시지 (질문 내용) AI 답변 생성
Google Search Google LLC 검색 질의 (채팅 내용 기반) 최신 대회 정보 검색 (Search Grounding)

위 서비스 외에 이용자의 개인정보를 제3자에게 제공하지 않습니다. 각 제3자 서비스의 개인정보 처리에 관한 사항은 해당 서비스의 개인정보처리방침을 참고하시기 바랍니다.

  • Supabase 개인정보처리방침
  • Google 개인정보처리방침

5. Your Rights — 사용자 권리

이용자는 언제든지 다음의 권리를 행사할 수 있습니다.

  • 개인정보 열람 요청: 수집된 본인의 개인정보를 확인할 수 있습니다.
  • 개인정보 수정 요청: 부정확한 정보의 수정을 요청할 수 있습니다.
  • 개인정보 삭제 요청: 수집된 개인정보의 삭제를 요청할 수 있습니다.
  • 계정 탈퇴: 앱 설정에서 직접 회원 탈퇴할 수 있으며, 탈퇴 시 모든 개인정보가 즉시 삭제됩니다.
  • 채팅 이력 삭제: 앱 내에서 개별 대화 또는 전체 채팅 이력을 직접 삭제할 수 있습니다.
  • 알림 수신 거부: 기기 설정 또는 앱 설정에서 푸시 알림을 해제할 수 있습니다.

위 권리 행사는 앱 내 설정 화면 또는 아래 연락처를 통해 요청할 수 있으며, 요청 접수 후 지체 없이(최대 10일 이내) 처리합니다.
