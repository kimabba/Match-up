## Executive Summary

### 1. 가장 중요한 변화 — 2025-12-12 KTA·KATO 랭킹 통합 협약

KTA(주원홍 회장) ↔ KATO 랭킹 통합 협약 체결. 생체 랭킹 + KATO 랭킹 + 대회요강·등급체계·상벌 통합. **KATA는 협약 미참여** → 향후 KTA-KATO vs KATA 양강 구도로 재편.

→ **Match-up은 통합 후 등급/랭킹 단일화를 가정해 설계**하는 것이 안전.

### 2. 단체별 위상 (동호인 체감)

| 단체 | 인지도 | 권위 | 주 참가자 | 비고 |
|------|--------|------|----------|------|
| **KATA** | 상 (양강) | 상 | 수도권 직장인·실력파 | **회원 절대수 최대** — 협약 빠짐 |
| **KATO** | 상 (양강) | 상 | 수도권+중부 직장인·실력파 | 가장 체계적, 5단계 선수 분리 시도 |
| **KTA 생체** | 중 → 상 (협약 후) | 중 → 상 | 시·군 클럽 | 공식성 압도적, 통합 후 헤게모니 |
| **KTFS** | 중 (지역 편차) | 지역 단위 상 | 지방 시·군 + 시니어 | 시·도 분권형, 가장 까다로움 |

### 3. 광주·전남 권역 핵심 사실

- **사실상 단일 권역**: 합동 상벌위원회(2023.10.03) + 공동 등급표("광주·전남 생활체육 랭킹관리규정") + 정기 합동 대회 다수
- **약 130개 클럽 + 동호인 15,000명** (디지털광주문화대전)
- **자체 부서명**: 골드부 / 남자일반부 / 여자금배부 / 여자신인부 / 국화부 / 지도자부 — KTA 표준(개나리/국화/청년/장년/베테랑)과 명칭 다름
- **자체 등급제 1~6급 + 신인부** — KTA 표준 1~5부보다 1단계 더 세분화
- **합동 대회 30~50개/년** 추정 — 빛고을배, 광주협회장배 광주·전남, 강진청자배, 해남땅끝배, 자평 상무수치과배, 영광 천년의 빛 등

### 4. 가장 큰 페인 포인트 (신규 동호인 입장)

1. **단톡 의존 구조** — 클럽 미가입자는 신청 마감조차 모름. **Match-up이 가장 큰 차별화 가능 영역**.
2. **부서·등급 매칭 어려움** — 협회마다 등급 이름·기준 다름.
3. **페어 구하기** — 클럽 미가입자는 페어 자체가 안 잡힘.
4. **샌드배깅(위장 등급)** — 모든 단체 공통 불만. 우승은 사실상 선수 출신.
5. **셀프콜 분쟁** — 협회마다 미스콜 인정 규정 다름 (KTFS 1회 인정, 다른 단체 0회).
6. **노쇼·늦참** — 페널티 규정 있으나 집행 느슨.

### 5. Match-up 우선 노출 단체 권고

1. **KATA + KATO** (양강, 동호인 핵심) — 디폴트 1·2순위
2. **KTA 생체** + 광역시·도협회 (공식성·시·군 단위) — 우선순위 3
3. **KTFS** 시·도 연합회 — 지방 사용자 메인 타깃 시 추가
4. **KSTF / KSSTA / KASTA** — 사용자 프로필 (60+ / 단식 선호)로 노출 토글

### 6. 데이터 모델 인사이트 (1차 권고 보강)

1차 보고서의 **옵션 C (절충)** 유지 + 다음 보강:
- `region_code` 컬럼 (권역, 검색 키)
- `tournaments.host_associations text[]` (예: `['광주협회','전남협회']` — 합동 대회)
- `tournaments.entry_fee_unit` 추가 (`per_team` / `per_person`) — 동호인은 99% per_team
- `tournaments.format` 분리 (`doubles` / `singles` / `mixed`) — 단식은 별도 카테고리
- KATA SSL 만료 → in-app webview 대신 외부 브라우저 처리

# Part A. 단체별 대회 위상·규모·문화 비교

## A.1 KTA 생활체육 부문 (대한테니스협회)

- **위상**: 협회 중 가장 공식적이지만 동호인 체감 인지도는 KATO/KATA보다 낮음. KTA = 엘리트 선수 중심 이미지. **2025-12 KATO 통합 협약으로 위상 급상승 중**.
- **회원 규모**: 시·군 협회 산하 등록 동호인. 한국 테니스 인구 약 90만(2025). KTA 정식 등록 동호인 통계 미공개.
- **대회 운영**: 시·도 협회장배 중심, 연간 수십~100+ (시·도 합산)
- **참가비**: 팀당 4~6만원 추정
- **상금**: 트로피 + 부상(테니스 용품). 현금 상금 작은 편
- **참가자**: 시·군 클럽 단위, 지역 색 강함
- **운영**: 셀프콜 + 본부 순회심판
- **인프라**: [join.kortennis.or.kr](https://join.kortennis.or.kr/) (통합시스템) + [kortennis.or.kr](https://www.kortennis.or.kr/)

**한 줄**: *공식성·시·군 단위 정통성*은 높지만 동호인 체감 인지도는 KATA/KATO에 밀림 — 다만 2025 KATO와 통합 후 헤게모니 단체로 부상 중.

## A.2 KATO (사단법인 한국테니스발전협의회)

- **위상**: 동호인 사이에서 **가장 권위 있는 두 단체 중 하나** (KATA와 양강). [클리앙 글](https://www.clien.net/service/board/cm_tennis/18598854): "랭킹 얘기하면 KATA·KATO".
- **회원 규모**: 미공개. 베테랑부 단체회원 연회비 30만원 ([KATO 규정](https://www.kato.kr/katoRules)).
- **대회 운영**: **연 30~50회**. 2025년 2~4월에만 8개 (대전·인천·천안·수원·창원·상주·부여 등) ([shiningsense 정리](https://shiningsense.com/3599)).
- **참가비**: **팀당 54,000원** (꿈나무 육성기금 4,000원 포함)
- **상금**: 부서·대회 등급에 따라 차등. 부상 = 테니스볼·스포츠타올·지역 특산물
- **참가자**: 수도권 + 충청북부권 + 경기권 직장인·실력파
- **운영**: 셀프콜 + 본부심판 호출. **32강 이상 출전해야 랭킹 포인트** 인정. 시드 1/4 제한, 선수 출신 5단계 분류 (Pure Amateur / Champion / Super-Grade / Instructor / Former Athletes)
- **신뢰성**: 규정·징계 가장 정형화. 3년 3회 징계 시 영구 출전 정지
- **인프라**: [kato.kr](https://kato.kr/) — UI 직관적

**한 줄**: *수도권·중부 중심 진지한 동호인 + 실력파*가 모이는 **체계적·권위적인 양강 중 하나**. 2025 KTA 협약으로 사실상 KTA 생체 = KATO 라인업으로 통합.

## A.3 KATA (한국동호인테니스협회 / Korea Amateur Tennis Association)

- **위상**: **"명실상부 국내 최대 규모의 동호인 테니스 단체"** (자체·블로그 표현). 30주년 시상식 약 300명 참석 ([Daum 재게시](https://v.daum.net/v/20250105185350014)). [노컷뉴스](https://m.nocutnews.co.kr/news/6075158)는 "韓 테니스 동호인 최강자"로 보도.
- **회원 규모**: 정확 비공개. 시상식 규모·자유게시판 활성도(6,000+ 게시글)로 KATO보다 회원 절대수 많은 것으로 추정.
- **대회 운영**: 연중 거의 매주 (50+/년). 바볼랏배·용인협회장배·수원화홍배·가평자라섬배 등.
- **참가비**: 팀당 5만원 내외 추정 (사이트 SSL 이슈로 직접 확인 미완)
- **참가자**: **수도권 집중**. 서울·경기 직장인 동호인 + 클럽 단위
- **운영**: 셀프콜 + 본부 호출. 베스트15제 (32강 이상 출전 대회 중 상위 15개 합산) — KATO와 거의 동일
- **명칭 혼선**: 자체 표기 "한국**동호인**테니스협회"(ikata 도메인) vs [테니스코리아 2025-12 기사](https://www.tennis.co.kr/news/articleView.html?idxno=51444)는 "한국테니스**진흥**협회"
- **인프라**: [new.ikata.org](http://new.ikata.org/) — **SSL 인증서 만료**. UI 2000년대 후반 풍

**한 줄**: *수도권 직장인·실력파* 중심의 **회원·대회 절대 수 최대** 단체 — KTA-KATO 통합 협약 미참여.

## A.4 KTFS (국민생활체육 전국테니스연합회)

- **위상**: KATA·KATO 대비 인지도 낮음. [클리앙](https://www.clien.net/service/board/cm_tennis/18598854): "수도권 외 지역은 대체로 KATO, KTFS 쪽 대회들이 많은 편".
- **회원 규모**: 시·도 연합회 합계 — 동호인 절대수 큼 (국민생활체육 산하)
- **대회 운영**: 시·도별 운영, 합산 시 **연간 가장 많을 가능성**. [경기도 연합회](http://www.kgtfs.com/)만 봐도 활발
- **참가비**: 추정 3~5만원/팀
- **상금**: 트로피·기념품 중심. 현금 상금 거의 없거나 소액
- **참가자**: **지방 시·군 클럽 + 시니어 비중 높음**. 지역 텃새 가장 강함
- **운영**: 셀프콜 + 클럽 단위. **시·도별 룰 차이**. 미스콜 1회만 인정 등 셀프저지 윤리규정
- **신뢰성**: 시·도 편차 큼. 운영자 역량에 따라 일정 변경·노쇼 빈도 차이
- **인프라**: 통합 사이트 부재. 시·도별 별도

**한 줄**: *지방 시·군 클럽 + 시니어*가 모이는 **지역 밀착형, 분권화된 네트워크** — 앱 입장에서 시·도별 어댑터 필요.

## A.5 시니어·사설 투어 (참고)

### KSTF (한국시니어테니스연맹)
- 남 60+, 여 50+ 대상. 60·65·70·75·80·85세부 5살 단위
- [kstf.kr](https://kstf.kr/) — 대회참가 신청

### KSSTA (한국슈퍼시니어테니스협회)
- KSTF와 별개. 충청북도 청주 소재
- 80+ 부서 미운영 시 75+ 은배 출전 허용 등 유연

### KASTA / 단테매 (단식 테니스 매니아)
- **단식 전용**. Daum 카페 단식 테니스 매니아 약 2.8만~3.2만 회원
- 한국 동호인 99%가 복식인 환경에서 단식 마니아 = **사실상 별도 카테고리**

## A.7 단체별 문화 차이 (카페·후기 인용)

- **선수 출신 vs 순수 동호인 분쟁**: 모든 단체 공통. [tenniseye](https://tenniseye.com/freeboard/117994): "16강 이상은 사실상 전직 선수, 순수 동호인 우승은 거의 불가능". KATO만 [규정상 5단계 분리](https://www.kato.kr/katoRules) 시도.
- **진지함**: 동호인 = "준선수" 분위기. [tenniseye](https://tenniseye.com/board_BVbH75/118205): "결승 새벽 1시 17분 종료" 등 장시간.
- **지역색**: 수도권=KATA + KATO, 수도권 외=KATO + KTFS
- **여성부 명칭 논쟁**: [오마이뉴스](https://www.ohmynews.com/NWS_Web/View/at_pg.aspx?CNTN_CD=A0003210589) — "개나리·국화" 명칭이 시대착오적이라는 비판 → 향후 변경 가능성

# Part B. 광주·전남 깊이 + 6개 권역 비교

## B.1 광주·전남 권역 (★ 시드 지역)

### B.1.1 운영 주체 계층

```
KTA (대한테니스협회, 통합)
  └─ 광주광역시테니스협회 (062-714-3244, 진월국제테니스장 지하1층)
        ├─ 동구/서구/남구/북구/광산구 — 5개 자치구 협회
        ├─ 지도자위원회
        └─ 클럽 130여 개, 동호인 약 15,000명
  └─ 전라남도테니스협회 (jntennis.co.kr)
```

- **광역 협회장배 / 전국 동호인**: **현금 시상 표준**
  - 구례 노고단배: 우승 120만, 준우승 80만
  - 일반 전국대회: 우승 50만 수준
- **KATO·KATA**: 부서별 차등. 마스터스 우승은 라켓 1자루(고급)+상품권

> **샌드배깅 비판**: tenniseye 카페 "선수 출신이 상금 가져가는 구조", "상금이 사실상 조직 경비로 환원" — 시상 투명성 신뢰 낮음

## C.5 매너·분쟁 트렌드

### 셀프콜 시스템
- 동호인 표준. **"자신 코트는 자신이 심판"**
- **미스콜 규정 단체별 차이**: KTFS는 미스콜 1회 인정 → let, 다른 단체는 0회 (분쟁 원인)

### 자주 발생하는 분쟁
1. **라인콜 분쟁** (가장 빈번)
2. **레트(let) 처리** (네트 인 시점·재경기)
3. **시간 끌기** (포인트 사이 25초 룰)
4. **공 정리 미흡** (자기 공 때문에 자기 실점)
5. **노쇼·늦참** (페널티 규정 있으나 느슨)
6. **선수 출신 출전 시비** (위장 등급 의혹)

### 분쟁 해결
- 1차: 선수끼리 합의
- 2차: 진행본부 호출 → 대회위원장 판정
- 3차: 재경기·몰수패. 비매너 행위 시 경고 → 점수 페널티 → 몰수패

### 관행 매너
- 시작·종료 시 네트 앞 악수
- 포인트 후 서버 기준 먼 사람부터 공 패스
- 상대 준비 안 되면 양손 들기
- 네트 인 시 "미안" 제스처
- 경기 후 공 회수, 라인 청소
- 라켓 던지기·욕설 금지
- 회식: 클럽 단위 참가 시 거의 필수 (술 강요 줄어드는 추세)

## C.6 동호인 문화 트렌드

- **2024 ITF**: 세계 테니스 인구 첫 1억 명 돌파. 한국 약 90만 (2024)
- **2028 예상**: 279만 명, 연 20% 성장
- **입문자 주 연령**: 20대 (Z세대)
- **여성 증가**: 동호회 활동 비율 남 4.5%(2019) → 6.8%(2024) (1.51배), **여 3.8% → 7.7% (2배+)**
- **대회 출전층**: 여전히 40~60대 남성 중심 (전통 클럽 카르텔)
- **클럽 평균**: 30~80명, 평균 50명. 큰 클럽(세이테니스) 1,200명
- **남녀 비율**: 전통 클럽 7:3, 신규 도시 5:5 근접
- **코로나 후 변화**:
  - 골프 대체로 MZ 유입 → 패션·라이프스타일 산업 동반 성장
  - 사설 매칭 플랫폼 폭발 (스매시, 코트스캐너, 어메추리그, 코트밸리)
  - **클럽 가입 없이 앱 매칭만으로 활동하는 신규 동호인 증가**
  - 호텔 테캉스 같은 럭셔리 시장
- **세대 단절**: 대회 참가층(40~60대) vs 신규 입문층(20~30대 여성). 신규층은 사설 매칭·미니 리그 선호

## C.7 디지털 격차

- 협회 사이트 갱신 빈도 천차만별. 모바일 미대응·HTTPS 미적용·HWP 첨부 다수
- KTA 통합시스템(`join.kortennis.or.kr`) — UI 진입장벽 → 동호인 사용률 낮음
- 신규 동호인 1차 접점: **인스타 → 네이버 카페 → 협회 사이트** 순

### 신규 동호인 정보 접근 장벽
1. 단톡방 없으면 신청 마감 모름
2. 부서·등급 매칭 어려움
3. 페어 구하기 (클럽 미가입자)
4. 입금 → 등록 확인 시차
5. 대진표 PDF·HWP 다운로드 (모바일 어려움)
6. 결과·랭킹 시스템 분절 (KATA/KATO/KSSTA 각각)

# Match-up 데이터 모델 권고 (수정판)

리서치 결과를 종합한 **모델 옵션 C′** (1차 보고서의 옵션 C 보강):

## 핵심 추가 컬럼

```sql
-- regions 테이블 (권역 매핑 — 신설 권장)
create table public.regions (
  code text primary key,                  -- 'gwangju_jeonnam', 'busan_ulsan_gn', 'seoul_metro', ...
  display_name_ko text,                   -- '광주·전남'
  governing_associations text[],          -- ['KTA-광주', 'KTA-전남']
  has_unified_ruleset boolean,            -- true (광주·전남), false (수도권)
  uses_kato boolean default false,
  uses_kata boolean default false,
  notes text
);

-- user_sports 보강
alter table public.user_sports
  add column tennis_org text,             -- 'kta' | 'kato' | 'kata' | 'ktfs' | 'kstf' | 'local'
  add column tennis_division_local text,  -- '골드부' / 'KATO 챌린져부' / '성남 은배부' 등 자유
  add column tennis_score numeric(3,1),   -- 1.0~10.0 (KTA 합산점) / 1~9 (제주)
  add column tennis_grade_expires_at date,
  add column region_code text references public.regions(code);

-- tournaments 보강
alter table public.tournaments
  add column region_code text references public.regions(code),
  add column host_associations text[],    -- ['광주협회','전남협회'] 합동 대회
  add column division_label_local text,   -- '남자 골드부' (광주 표시용)
  add column division_kta_standard text,  -- KTA 표준 매핑 (검색용)
  add column entry_fee_unit text default 'per_team',  -- 'per_team' | 'per_person'
  add column is_joint_event boolean default false;

-- 등급 별칭 dictionary (선택)
create table public.tennis_grade_aliases (
  region_code text references public.regions(code),
  grade text,                              -- 'rookie', 'div3' 등 (현재 enum)
  display_label text,                      -- '신입부' (광주) / '청년부 신인' (서울)
  primary key (region_code, grade)
);
```

## 콘텐츠 작성 우선순위 (rule_articles)

1. **광주·전남 권역** (시드 지역) — 부서 매핑, 합동 운영, 가입 절차

3. **KATO·KATA 양강 비교** — 동호인 의사결정 가장 도움
4. **셀프콜·매너** — 신규 동호인 페인 직결 (RAG에서 빛 발하는 영역)
5. **수도권** (KATA 비중 + 다중 등록 패턴)
6. **나머지 권역** (영남·충청·강원·제주)
